# Personalised Treatment Recommendation using Reinforcement Learning

## Project Description
This project uses Reinforcement Learning (RL) to recommend personalized treatments for patients based on medical data.  
The system learns optimal treatment strategies by analyzing patient states and rewards.

## Technologies Used
- Python
- Reinforcement Learning
- Machine Learning
- NumPy
- Pandas
- Matplotlib

## Features
- Patient data analysis
- RL-based treatment recommendation
- Visualization of learning results
- Performance evaluation
